Complete steps to follow to download the Jarvis on your system:

## Prerequisites

1\. Download the project zip or clone the repository.

2\. (Windows) Download MSVC Build Tool - https://aka.ms/vs/17/release/vs_BuildTools.exe  
    Select: Desktop development with C++

3\. Install Python 3.11-3.13 and Node.js v20.19.6 (the bootstrapper enforces this Python range and the exact Node.js version):
    - Python download (Windows) - https://www.python.org/downloads/windows/
    - Node.js download (v20.19.6) - https://nodejs.org/dist/v20.19.6/

## Run (Windows)

1\. Extract the folder and open Command Prompt in the repo root.

2\. Run the bootstrapper with Python 3.11-3.13:

```
assistant.bat start
```

> Tip: You can also run diagnostics with `assistant.bat doctor`.

## Run (macOS/Linux)

1\. Open a terminal in the repo root.

2\. Run the bootstrapper:

```
python3 cli/bootstrap.py start
```
